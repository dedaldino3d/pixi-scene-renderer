Pixi Renderer Pro i Pixi Renderer Pro is a high-performance, modular 2D scene renderer built for Creative Flow.

Purpose: Demonstrate advanced, GPU-accelerated vector rendering, boolean operations, and real-time effects using PixiJS v8.

Features: Parse and render JSON scenes, support for boolean ops (union/intersect/difference), edge detection, PNG export, and 60 FPS with 1000+ objects.

For Creative Flow: This project showcases scalable, interactive graphics technology for creative and design applications.
